# Examples

## Basic table

Pass `data` and `config` to table to generate a table. The example below uses the minimal configuration needed to generate a table with static data.

<Story name="Basic table"
       args={{
         data: [
           {
             firstName: 'Mark',
             lastName: 'Parker',
             gender: 'male',
             favoriteFood: 'Pasta',
           },
           {
             firstName: 'Mary Jane',
             lastName: 'Watson',
             gender: 'female',
             favoriteFood: 'Pizza',
           },
         ]
       }}>
  {Simple.bind({})}
</Story>

## Custom templates
