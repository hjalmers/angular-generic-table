Styling

Columns and rows

Since generic table uses standard html elements like tr, td, th etc. you can style it like you normally would style a table. In addition generic table supports conditional classes for both rows (tr elements) and individual columns (td elements).

Pass either a function or a string to rowClass and/or columnClass in field declaration to add custom classes to rows and/or columns.

The example below is not intended to be realistic or beautiful, it's just meant to show some of the things you can do with rowClass and columnClass. Note that th is passed as row property for the table header and total for total rows, something that's handy if you just want to apply styles to data rows.

Table element

Use gtClasses input to append class names to the table element.

Using Bootstrap?

Bootstrap has a couple of pretty useful utility classes which works great with generic table, for example add the class text-right to columnClass to align the content of the column to the right, or maybe d-none d-md-table-cell to hide a column on small screens.
